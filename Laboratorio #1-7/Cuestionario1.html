Cuestionario #1

1) ¿Qué es una base de datos?

  • Es una colección de archivos interrelacionados que permite guardar/organizar/recuperar información con la menor redundancia posible.

2) ¿En qué casos es conveniente usar bases de datos?

  • Cuando la información está presente en varios archivos relacionados
  • Cuando la información es requerida por varios usuarios/aplicaciones
  • Cuando el modelo de datos es muy complejo y necesita varios archivos
  • Cuando los archivos tienen un volumen de datos (registros) muy grande

3) ¿Qué es un sistema de gestión de base de datos (DBMS)?

  • Conjunto de programas que gestionan grandes bloques de información con el fin de proporcionar un entorno eficiente para extraer/almacenar información de una base de datos.

4) Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • Satisfacer restricciones de consistencia en los datos
  • Controlar la interacción entre usuarios concurrentes
  • Reestablecer la base de datos antes de que ocurra un fallo
  • Limitar el acceso de los usuarios al contenido de la base de datos

5) Ejemplifica y justifica en qué proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

  • Hubiera sido interesante usar una base de datos en nuestro proyecto de Videojuegos para guardar las puntaciones de los jugadores. Esto habría incentivado a los usuarios a jugar más partidas para obtener una mejor posición en los marcadores.
