Laboratorio #1

1) Descripción de tecnologías

  • URL = Es la dirección específica que se le asigna a una página web. Cadena de caracteres que permite identificar y recuperar recursos de Internet (1).
  • MIME = Estándar que indica el formato y tipo de información de un archivo que se envía a través de la red (2).
  • Códigos de errores HTTP = Códigos que indican que una petición HTTP fue incorrectamente solicitada por el usuario (3).
  • XML = Lenguaje de mercado que permite compartir y almacenar datos entre diferentes sistemas de manera legible (4).
  • XHTML = Lenguaje de mercado que permite expresar HTML como XML válido (5).
  • HTML5 = Quinta versión de HTML que mejora la estructuración de documentos HTML e interacción con contenido multimedia (6).
  • Javascript = Lenguaje interpretado y orientado a objetos que implementa funcionalidad a los elementos de una página web (7).
  • PHP = Lenguaje de código abierto que permite generar páginas web dinámicas mediante un servidor (8).
  • JSP = Tecnología que permite desarrollar páginas web ejecutables en varios servidores y plataformas (9).
  • AJAX = Técnica de desarrollo web que permite generar páginas capaces de actualizarse de forma continua (10).

2) ¿Cuál es la diferencia entre Internet y la World Wide Web?

  • Internet = Red de redes que permite la interconexión de computadoras a nivel mundial (11).
  • World Wide Web = Sistema interconectado de páginas web accesibles a través de Internet (12).

3) ¿Cuál es el propósito de los métodos HTTP? (13)

  • GET = Solicita la representación de un recurso específico. Sólo recupera datos.
  • HEAD = Obtiene los headers de un recurso solicitado mediante un GET.
  • POST = Envía información hacia un recurso específico del servidor.
  • PUT = Crea un nueva recurso o actualiza la representación de uno existente.
  • DELETE = Borra un recurso en específico.
  • PATCH = Actualiza un campo específico de un recurso.

4) ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué? (13)

  • POST = Permite enviar información al servidor para acceder a un recurso

5) ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? (13)

  • GET = Permite obtener información del servidor

6) Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error? (3)

  • La petición HTTP del usuario fue completada exitosamente.

7) ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué? (3)

  • Error 404: El servidor no encontró el recurso solicitado
  • Sí, porque significa que la conexión hacia la página web no fue configurado correctamente.

8) ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué? (3)

  • Error 500: El servidor no pudo resolver una petición
  • Sí, porque significa que hay un evento en la página web que el servidor no puede procesar

9) ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados. (14)

  • Depreciado = La versión actual de HTML ya no soporta la funcionalidad del atributo puesto que fueron reemplazados por otros más eficientes
  • Lista de atributos depreciados:

    • acronym applet basefont big center
    dir font frame frameset isindex
    noframes s strike tt u

10) ¿Cuáles son las diferencias principales entre HTML4 y HTML5? (14)

  • HTML5 soporta contenido multimedia
  • HTML5 simplifica la sintaxis de los atributos
  • HTML5 facilita el uso de atributos sin usar div
  • HTML5 permite incorporar CSS y Javascript por separado
  • HTML5 ignora el uso de mayúsculas-minúsculas en los atributos

11) ¿Qué componentes de estructura y estilo tiene una tabla? (15)

  • table = Crea la tabla
  • thead = Encabezado de la tabla
  • tbody = Cuerpo de la tabla
  • th = 1° fila de la tabla
  • tr = Filas de la tabla
  • td = Celdas de la tabla
  • colspan = Crea varias columnas
  • rowspan = Crea varias filas
  • border = Define el borde de las celdas
  • border-collapse = Fusiona los bordes
  • padding = Añade espaciado al texto de una celda
  • text-align = Define la posición del texto en una celda
  • border-spacing = Añade espaciado entre celdas

12) ¿Cuáles son los principales controles de una forma HTML5? (16)

  • Input = ¿Qué tipo de formulario se desea rellenar?
  • Action = ¿Qué se realiza cuando se sube un formulario?
  • Target = ¿El formulario abrirá una nueva pestaña?
  • Method = ¿Qué método HTTP se usa para subir el formulario?
  • Name = ¿Cuál es el nombre del formulario?
