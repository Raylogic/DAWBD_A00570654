Preguntas:

1) ¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)? (1)

  • Asegura que una aplicación web trabaje de forma incondicional en un navegador, tal como una aplicación de escritorio
  • Reduce el tiempo de procesamiento de tareas entre cliente-servidor

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor? (2)

  • SQL Injection, XSS, Vulnerabilidades XMLHTTP Request
  • Las validaciones se hacen del lado del cliente antes de enviarlas al servidor
  • JQuery Validate es una biblioteca que permite validar formularios del lado del cliente

3) ¿Qué es JSON? (3)

  • Es un formato basado en texto que representa datos estructurados en la sintaxis de objetos de Javascript
  • Permite intercambiar-transmitir datos en aplicaciones web
