Preguntas:

1) Explica y elabora un diagrama sobre cómo funciona AJAX con jQuery.

  • JQuery permite usar varios métodos de AJAX para solicitar texto, HTML, XML o JSON para cargar datos externos directamente en un HTML. En un sistema, el servidor web con la base de datos (backend) se comunica con el HTML y CSS (frontend) mediante el control de AJAX para pedir-solicitar información. JQuery se encarga de procesar las peticiones y desplegarlas en el HTML a través de métodos AJAX.

2) ¿Qué alternativas a jQuery existen? (2)

  • JS - Librería nativa de Javascript
  • UmbrellaJS - Métodos de Javascript para selectores
  • MooTools - Framework de Javascript
  • ZeptoJS - Librería minimalista de Javascript
  • ExtJS - Javascript Extendido basado en YahooUserInterface
