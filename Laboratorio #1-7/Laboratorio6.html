1) ¿Cuáles son las diferencias entre los posibles valores de la propiedad position? (1)

  • Absolute = El elemento se posiciona con relación al elemento padre
  • Static = El elemento se posiciona en función del orden del HTML
  • Fixed = El elemento se posiciona con respecto a la ventana
  • Relative = El elemento se posiciona con respecto a su posición normal
  • Sticky = El elemento se posiciona con respecto al scrolling del usuario

2) ¿Cuáles son los valores estándar para la propiedad visibility? (2)

  • Default value = visible
  • Inherited = yes
  • Animatable = yes
  • Syntax = object.style.visibility=""

3) ¿Qué es el zIndex y para qué sirve? (3)

  • Propiedad CSS que specifica el orden de apilamiento de los elementos
