1) ¿Qué hace la función phpinfo()? Describe 3 datos que llamen tu atención. (1)

  • Comando que muestra la configuración de PHP. Algunos de los más intersantes son las variables relativas al servidor, las funciones asociadas al HTML, los protocolos de red que puede manejar y el soporte de DOM.

2) ¿Qué cambios tendrías que hacer en la configuración del servidor para que pudiera ser apto en un ambiente de producción? (2)

  • Instalar una aplicación de entorno de servidor como XAMPP, la cual incluye Apache, MySQL y PHP para poder usar y manejar el servidor.

3) ¿Cómo es que si el código está en un archivo con código HTML que se despliega del lado del cliente, se ejecuta del lado del servidor? (3)

  • Un link en realidad realiza una petición a un archivo HTML encontrado en el servidor para que este lo envíe a nuestro navegador y lo ejecute.
