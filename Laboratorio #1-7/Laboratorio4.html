Laboratorio #4

1) ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Javascript Java
1) Lenguaje de scripts 1) Lenguaje de programación
2) Lenguaje interpretado 2) Lenguaje compilado e interpretado
3) Alcance basado en funciones 3) Alcance basado en clases
4) Paradigma basado en prototipos 4) Paradigma basado en objetos
5) Lenguaje débilmente tipado 5) Lenguaje fuertemente tipado
6) Concurrencia basada en eventos 6) Concurrencia basada en threads
7) Usa sólo tipos de dato flotantes 7) Soporta varios tipos de datos
8) Basado en la compatibilidad del navegador 8) Plataforma independiente
Ambos usan lenguajes orientados a objetos

2) ¿Qué métodos tiene el objeto Date? (Menciona 5)

  • getTime() = Devuelve los milisegundos entre una fecha y el 01/01/1970
  • getDay() = Devuelve el día de la semana
  • getDate() = Devuelve el día del mes
  • getFullYear() = Devuelve el año completo
  • getTimezoneOffset() = Devuelve los minutos entre tiempo local y UTC

3) ¿Qué métodos tienen los arreglos? (Menciona 5)

  • pop() = Elimina el último elemento del arreglo
  • push() = Añade un elemento al final del arreglo
  • shift() = Elimina el primer elemento del arreglo
  • unshift() = Añade un elemento al inicio del arreglo
  • slice() = Extrae una sección del arreglo en uno nuevo

5) ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  • Como pueden ser accedidas por cualquier función, no es recomendable usarlas como argumentos para las mismas excepto cuando haya alguna dependencia de valores entre ellas.

6) ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

  • match() = Devuelve las coincidencias de una expresión regular en un string.
  • Puede usarse con una Search Box para buscar archivos o palabras clave en una página web.
