Preguntas:

1) ¿Por qué es una buena práctica separar el controlador de la vista? (1)

  • Facilita la mantenibilidad del sistema, reutilización de código y separación de conceptos.

2) Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función? (2)

  1. $GLOBALS = Tiene una referencia a cada variable del alcance del script.
  2. $_SERVER = Contiene la localización de los scripts de un programa.
  3. $_FILES = Guarda los elementos subidos al script via el método POST.
  4. $_REQUEST = Guarda el contenido de $_GET, $_POST y $_COOKIE.
  5. $_COOKIE = Guarda las variables enviadas al script vía HTTP cookies.
  6. $_SESSION = Guarda las variables de sesión del script actual.
  7. $_PHP_SELF = Guarda el nombre del script de PHP llamada.

3) Explora las funciones de php, y describe 2 que no hayas visto en otro lenguaje y que llamen tu atención (3)

  • base_convert() = Convierte un número de una base numérica a otra
  • array_multisort() = Ordena arreglos multidimensionales
  • str_word_count() = Cuenta el número de palabras en un string

4) ¿Qué es XSS y cómo se puede prevenir? (4)(5)

  • Vulnerabilidad que permite inyectar secuencias de comandos de Javascript en páginas web
  • Método de prevención: Usar el método htmlspecialchars() convierte determinados caracteres en entidades HTML para bloquear su uso en comandos.
